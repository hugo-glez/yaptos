This program aim's to create a swf file from a serie of images.
The first parameter to the program is the folder where the jpg files are, and the second parameter is the name of the output swf file.

You need 4 images for every button, that the names are:
UP		OVER		HIT		DOWN
next0.jpg	prev0.jpg	last0.jpg	first0.jpg
next1.jpg	prev1.jpg	last1.jpg	first1.jpg
next2.jpg	prev2.jpg	last2.jpg	first2.jpg
next3.jpg	prev3.jpg	last3.jpg	first3.jpg

There are for the different states of the buttons, may you prefer the same image, you need to copy the same image for every name.

Enjoy it ..
